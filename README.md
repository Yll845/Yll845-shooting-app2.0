**Welcome to your standalone ShootingKS app** 

**About**

This project is configured to run locally without requiring Base44 services.

**Edit the code in your local development environment**

1. Clone the repository using the project's Git URL
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Run the app: `npm run dev`

No Base44 environment variables are required.

**Docs & Support**

Use the project source and local browser console for debugging.
